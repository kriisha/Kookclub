# Kookclub

Een zorgvuldig samengestelde collectie van 50 gerechten in het Nederlands. Filter op koolhydraten, proteïne, bereidingswijze en calorieën. Pas porties aan met een slider, en genereer automatisch een gecategoriseerde boodschappenlijst van alle geselecteerde gerechten.

## Stack

- **Next.js 14** (App Router)
- **React 18** + **TypeScript**
- **Tailwind CSS** voor styling
- **Framer Motion** voor animaties
- **Lucide React** voor iconen

## Features

- 50 gerechten met ingrediënten en stap-voor-stap instructies
- Filteren op koolhydraten, proteïne, bereidingswijze en max. calorieën
- Dynamische portie-slider (1-12 personen) die ingrediënten herschaalt
- Boodschappenlijst die automatisch ingrediënten aggregeert per categorie
- Voortgang bijhouden met afgevinkte stappen en ingrediënten
- Volledig mobile-first design
- LocalStorage persistentie voor je lijst

## Lokaal draaien

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
